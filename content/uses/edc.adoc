=== Phone - Google Pixel 9a

Running https://grapheneos.org/[GrapheneOS] with https://kvaesitso.mm20.de/[Kvaesitso] as a launcher. +
I use https://arcticons.com/[Arcticons] + https://f-droid.org/packages/com.kaanelloed.iconeration/[Alembicons] for icons. +

* https://keepassxc.org/[Keepass (XC)] for password management.
* https://organicmaps.app/[Organic maps] for navigation.
* https://signal.org/nl/[Signal] (with https://molly.im/[Molly]) and https://threema.com/en[Threema] for messaging.
* https://immich.app/[Immich] for photo storage and browsing.
* https://antennapod.org/[Antennapod] for listening to podcasts.
* https://github.com/polymorphicshade/Tubular[Tubular] for watching videos.
* https://github.com/breezy-weather/breezy-weather[Breezy weather] for weather forecasts.
* https://www.davx5.com/[DAVx5] and https://icsx5.bitfire.at/[ICSx5] for calendar and tasks synchronization.
* https://syncthing.net/[Syncthing] with https://github.com/researchxxl/syncthing-android[Syncthing fork] for file synchronization.
* https://catima.app/[Catima] for loyalty cards and tickets.
